# Christos face swapper

## Instructions (OSX)

* `brew tap homebrew/science`
* `brew install open-cv`
* `nvm use v7.5.0`
* `npm install`
* `npm install -g nodemon`
* `npm run start-dev`

## Config

###Environment variables

* `S3_ACCESS_KEY_ID`
* `S3_SECRET_ACCESS_KEY`
* `S3_BUCKET_NAME`
* `S3_REGION`
